Overview

This is a text labeling tool prototype showing basic abilities of nlp frameworks which could be used for automated text annotation.

Backend of the application is built on python with the help of spaCy - Natural Language Processing framework and flask - micro web framework

Frontend layout is made with https://getbootstrap.com CSS framework, interaction with api for text labeling is performed via AJAX requests

Annotation types:

Named Entity Recognition (NER)

NER is a subtask of natural language processing (NLP) that focuses on identifying and classifying named entities mentioned in unstructured text into predefined categories. Named entities are real-world objects that have proper names, such as:

  • Organizations: Names of companies, institutions, or groups (e.g., "Google," "United Nations")
  • Locations: Geographical locations, such as countries, cities, and landmarks (e.g., "Paris," "Mount Everest")
  • Dates and Times: Specific dates, times, or durations (e.g., "January 1, 2024," "two hours")
  • Monetary Values: References to currency amounts (e.g., "$100," "â‚¬500")
  • Persons: Names of people (e.g., "Albert Einstein," "Marie Curie")
  • Percentages: Percentage values mentioned in the text (e.g., "20% growth")
Linguistic annotations

These annotations are crucial for natural language processing (NLP) and computational linguistics because they help in understanding, processing, and analyzing language at various levels, such as syntax, semantics, phonetics, and more.

Linguistic annotations can range from simple labels indicating parts of speech to complex structures representing syntactic or semantic relationships.
